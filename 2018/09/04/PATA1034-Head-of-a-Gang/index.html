PATA1034 Head of a Gang

发表于 2018-09-04 | 分类于 PAT

One way that the police finds the head of a gang is to check people’s phone calls. If there is a phone call between A and B, we say that A and B is related. The weight of a relation is defined to be the total time length of all the phone calls made between the two persons. A “Gang” is a cluster of more than 2 persons who are related to each other with total relation weight being greater than a given threthold K. In each gang, the one with maximum total weight is the head. Now given a list of phone calls, you are supposed to find the gangs and the heads.

Input Specification:

Each input file contains one test case. For each case, the first line contains two positive numbers N and K (both less than or equal to 1000), the number of phone calls and the weight threthold, respectively. Then N lines follow, each in the following format:

where Name1 and Name2 are the names of people at the two ends of the call, and Time is the length of the call. A name is a string of three capital letters chosen from A-Z. A time length is a positive integer which is no more than 1000 minutes.

Output Specification:

For each test case, first print in a line the total number of gangs. Then for each gang, print in a line the name of the head and the total number of the members. It is guaranteed that the head is unique for each gang. The output must be sorted according to the alphabetical order of the names of the heads.

题目大意：
给出N条通话记录，相互有通话记录的人则联系在一起称为一个团伙，如果此团伙超过2人，并且总通话时长超过K，则称这个团伙为帮派（Gang)，其中通话时间最长者为帮派头头（head），求图中的帮派数量和各帮派头目
思路：
由于每个人的name由字符串给出，所以需要进行对数值下标的转换，最后又需要输出字符串，可以用map来进行相互映射。其他部分即可结合图及其遍历中给出的DFS伪代码进行相关功能的添加。此题DFS函数中需携带的变量除了当前访问结点下标外还需记录head,numM,totalValue(头目，成员数，总边权)
注意事项：
· 图的总结点数numP初始设为0，在输入过程中，遇到新名字则+1，由此获得~
· 这是无向图，接听和打出时间都算个人通话时间
· 题目中N不是结点数而是记录条数，N条记录最多对应2N个结点，所以数组长度maxn需开至2000以上！否则有一个测试点段错误~
