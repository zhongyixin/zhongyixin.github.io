STL库相关——变长数组vector

发表于 2018-09-06 | 分类于 STL |

STL:即C++标准模板库(Standard Template Library)
此篇为vector的常见用法整理。

头文件

#include
using namespace std;

初始化

(1)vector v; //空
(2)vector v(10); //定义了10个整型元素的向量
(3)vector v(10,1); //定义了10个整型元素的向量,且给出每个元素的初值为1
(4)int b[7]={1,2,3,4,5,9,8};
vector a(b,b+7); //从数组中获得初值b[0]-b[6]
数组也可以是vector c[n]
则复制变长数组c到a为vector a(c,c+n);

常用操作

v.push_back(x); //在尾部添加一个元素
v.pop_back(); //在尾部删除元素
v.size(); //获得vector中元素个数
v.clear(); //清空
v.empty(); //判断是否为空
v.insert(v.begin()+1,5); //在第一个元素（从0算起）的位置插入数值5，如1234变15234
v.resize(n); //将v的现有元素调整至n个
v.erase(v.begin()+3); //删除第三个元素
v.erase(v.begin()+1,v.begin()+4); //删除第1到第3个元素（左闭右开）
vector之间可以直接复制或比较操作
a.swap(b); //将向量a，b进行整体性交换
