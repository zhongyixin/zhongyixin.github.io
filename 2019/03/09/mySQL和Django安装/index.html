3.接下来启动服务和登陆
我进行得超级不顺利TT。在安装完MySQL后输入”net start mysql”后频繁提示”‘net’不是内部或外部命令，也不是可运行的程序或批处理文件”。先是因为PATH没设置好，再是要用管理员模式运行等等乱七八糟。总之在我的电脑->右键->属性->高级->环境变量->系统变量->Path中添加mysql的bin路径，然后用管理员模式进入DOS系统（就是在开始搜索框搜cmd然后右键打开它）
执行命令 net start mysql
成功启动后执行命令mysql -u root -p （password就回车，因为没设置） 出现mysql>就ok
